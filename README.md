##Run Code
1. Clone Repository
```https://github.com/ayushkaneria11/advisor-api.git```
2. Install All dependensise
```pip install -r requirements.txt```
3. Migrate Database
```python manage.py makemigrations```
```python manage.py migrate```
4. Run server
```python manage.py runserver```
